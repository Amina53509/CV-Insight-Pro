# 📄 ResumeVision AI — Smart AI Resume Analyzer

ResumeVision AI is an intelligent **AI-powered Resume Analyzer** built with **Python + Gradio**.  
It helps users analyze their CVs, detect skills, estimate experience level, suggest missing skills, and visualize resume quality using graphs.

## 🚀 Live Demo
👉 Upload your resume and get instant AI feedback (after deployment on Hugging Face Spaces)

## 🧠 Features

✔ Resume Score (0–100)  
✔ Email Detection (Valid / Suspicious)  
✔ Phone Number Extraction  
✔ Skill Extraction using AI logic  
✔ Missing Skill Suggestions (based on career field)  
✔ Career Field Detection (AI, Doctor, Teacher, Developer, etc.)  
✔ Experience Level Detection  
✔ Resume Improvement Tips  
✔ Resume Preview  
✔ Graphical Dashboard (Matplotlib Bar Chart)

## 📊 Supported Resume Analysis

The system detects and evaluates resumes for:

- 💻 Software Engineers
- 🤖 AI / ML Engineers
- 🌐 Web Developers
- 📊 Data Analysts
- 🏥 Doctors
- 📚 Teachers
- 💼 Accountants
